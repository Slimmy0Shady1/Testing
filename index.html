HLFEval: Hybrid LLM-Based Framework for Sentence-Level Evaluation

2025 - Present

Developed a hybrid framework for evaluating LLM-generated responses across semantic relevance, fluency, safety, and factuality. The framework combines metric computation, LLM-as-a-Judge, and learned weighting into one composite score.

Block A: Data preprocessing and input construction.

Block B: Metric computation for quality dimensions.

Block C: Weight learning and final score generation.
